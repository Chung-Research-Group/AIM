AIM documentation
=================

**AIM (Adsorption Integrated Modules)** is an open-source suite of MATLAB
graphical applications for adsorption isotherm analysis and fixed-bed process
modelling. The four modules form one workflow, from pure-component data to
multicomponent breakthrough predictions.

Choose a path
-------------

* **Run the Windows application:** follow :doc:`installation` and use the
  bundled installer.
* **Work from MATLAB source:** clone the repository, open ``AIM_Build.prj``,
  and review :doc:`development`.
* **Reproduce the publication cases:** start with :doc:`examples`.
* **Understand a module:** see :doc:`modules`.

.. note::

   The Windows standalone application uses MATLAB Runtime and does not require
   a MATLAB license. Running or modifying the source requires MATLAB R2024a or
   newer.
