Video tutorials
===============

These walkthroughs cover the complete graphical workflow for each AIM module.
The videos are streamed from Dropbox; a direct link is provided below each
player if embedded playback is blocked by the browser or network policy.

.. _IsoFit-label:

IsoFit tutorial
---------------

Import a single-temperature isotherm, select and fit a model, inspect the fit,
and export parameters for downstream modules. See also :doc:`isofit` and
:doc:`input-output`.
