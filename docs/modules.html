Module workflow#

AIM’s four modules exchange human-readable data files so a fitted equilibrium model can move directly into mixture prediction and process simulation.

1. Fit equilibrium data

IsoFit estimates parameters from a single-temperature isotherm.

IsoFit
2. Add temperature dependence

HeatFit fits multi-temperature data and isosteric heat.

HeatFit
3. Predict mixtures

MixPred applies EDSL or IAST to pure-component models.

MixPred
4. Simulate the bed

BreakLab couples equilibrium, kinetics, transport, pressure drop, and energy balances.

BreakLab

The Input and output data files guide defines the shared file formats and metadata. The Video tutorials page provides a complete video for each module.
